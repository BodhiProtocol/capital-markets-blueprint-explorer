# Capital Markets Blueprint Explorer

**50 Capital Markets concepts, one Lighthouse Blueprint each. Browse by season, search, dark/light mode.**

Live demo → https://bodhiprotocol.github.io/capital-markets-blueprint-explorer/

---

## Why this exists

Capital markets is usually taught as disconnected topics — instruments here, clearing there, risk somewhere else. Project Lighthouse reframes each concept as a **Blueprint**: a visual-metaphor card with a core question, one per concept, grouped into seasons that build on each other.

This tool makes all 50 capital markets Blueprints browsable and searchable in one place — a companion reference for anyone working through the curriculum.

## What's inside

- All 50 Capital Markets Blueprints, grouped by Season
- Search across every Blueprint's ID, title, and core question
- Season-filter pills to browse one season at a time
- Click any card to reveal its core question

## Tech

One self-contained `index.html`. No frameworks, no build step, no API calls. Dark/light theme toggle (light palette sourced from the "AI in One Picture" infographic). Data lives in a single in-file JS array sourced from the curriculum docs — no database, manual re-sync only.
